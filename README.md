# Nasals analysis

This script automatically extracts acoustic and spectral measures related to vowel nasality and voice quality from labeled speech data (TextGrids and sound files).

The script processes each labeled interval and computes temporal, spectral, and harmonic-based features, including formant values, harmonic amplitudes, and nasalance-related metrics. It enables consistent measurement across tokens and speakers, facilitating quantitative comparison within and across datasets. 
# Input
wav and TextGrid if avaliable.

# Usage
1. Download the file `nasals_acoustic_analysis.praat` by clicking the name of the file and then the green download button in the upper part of the code.
2. Open **Praat**.
3. In the *Praat Objects* window, choose **Open > Read from file...**.
4. Look for the file **nasals_acoustic_analysis.praat** and select it.
5. Once the script window opens, click **Run > Run** to execute the script.

## Column Descriptions

- **Filename** — Name of the analyzed audio file.
- **Interval** — Number of the interval in the TextGrid.
- **Interval_label** — Label assigned to the interval (e.g., isually IPA symbol).
- **duration_interval** — Total duration of the interval (in seconds).
- **duration_chunk** — Duration of the analysis window analysis inside the interval (in seconds).
- **f0_mean** — Mean fundamental frequency (Hz).
- **F1** — First formant frequency (Hz).
- **F2** — Second formant frequency (Hz).
- **A1** — Amplitude of the strongest harmonic near F1 (dB).
- **P0** — Amplitude of the low-frequency nasal peak below F1 (dB).
- **A1–P0** — Difference between A1 and P0; used as an estimate of vowel nasality.
- **H1_dB** — Amplitude of the first harmonic (H1) in dB.
- **H2_dB** — Amplitude of the second harmonic (H2) in dB.
- **H1–H2** — Difference between H1 and H2 amplitudes (dB).
- **H1–A1** — Difference between H1 and A1 amplitudes (dB).
- **spectral_peak** — Frequency of the main spectral peak (Hz).
- **f0_floor** — Lower bound used for F0 extraction (Hz).
- **f0_ceiling** — Upper bound used for F0 extraction (Hz).

## Notes

- A1–P0 and related measures should be interpreted relatively, by comparison with other tokens, rather than in isolation. Low A1–P0, higher nasalance.
- This script is intended for teaching use. It has been used in classroom settings, but the extracted measures and parameter settings have not been systematically validated. Users are therefore strongly advised to ensure they fully understand the meaning of each parameter and to carefully review the frequency bands defined for A1 and harmonic detection before applying the script to research data.
